# Aze Apps LLC Website Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and deploy the Aze Apps LLC company website (3 pages: Home, About, Apps) with Next.js on Vercel.

**Architecture:** Next.js 15 App Router with static export. Shared layout with navbar and footer. Tailwind CSS for styling with cream/navy color scheme. GitHub repo for source, Vercel for deployment.

**Tech Stack:** Next.js 15, TypeScript, Tailwind CSS, GitHub, Vercel

**If Tailwind v3** (tailwind.config.ts exists): Update `tailwind.config.ts`:

```typescript
import type { Config } from "tailwindcss";

const config: Config = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  theme: {
    extend: {
      colors: {
        navy: {
          DEFAULT: "#1B2A4A",
          light: "#2A3F6A",
        },
        cream: {
          DEFAULT: "#FDF6E3",
        },
      },
      fontFamily: {
        sans: ['var(--font-inter)', 'system-ui', 'sans-serif'],
        serif: ['var(--font-playfair)', 'Georgia', 'serif'],
      },
    },
  },
  plugins: [],
};

export default config;
```

And update `src/app/globals.css`:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

**If Tailwind v4** (no tailwind.config.ts, or CSS-based config): Replace `src/app/globals.css`:

```css
@import "tailwindcss";

@theme {
  --color-navy: #1B2A4A;
  --color-navy-light: #2A3F6A;
  --color-cream: #FDF6E3;
  --font-sans: var(--font-inter), system-ui, sans-serif;
  --font-serif: var(--font-playfair), Georgia, serif;
}
```

## Chunk 2: Shared Components

### Task 3: Build Navbar component

- [ ] **Step 1: Create `src/components/Navbar.tsx`**

```tsx
"use client";

import Link from "next/link";
import Image from "next/image";
import { useState } from "react";

export default function Navbar() {
  const [isOpen, setIsOpen] = useState(false);

  const links = [
    { href: "/", label: "Home" },
    { href: "/about", label: "About" },
    { href: "/apps", label: "Apps" },
  ];

  return (
    <nav className="bg-cream border-b border-navy/10">
      <div className="max-w-5xl mx-auto px-4 py-4 flex items-center justify-between">
        <Link href="/" className="flex items-center gap-2">
          <Image src="/logo.jpeg" alt="AZE Apps" width={40} height={40} className="rounded" />
          <span className="font-serif text-navy text-xl font-bold">AZE Apps</span>
        </Link>

        {/* Desktop nav */}
        <div className="hidden md:flex gap-6">
          {links.map((link) => (
            <Link
              key={link.href}
              href={link.href}
              className="text-navy hover:text-navy-light font-medium transition-colors"
            >
              {link.label}
            </Link>
          ))}
        </div>

        {/* Mobile hamburger */}
        <button
          className="md:hidden text-navy"
          onClick={() => setIsOpen(!isOpen)}
          aria-label="Toggle menu"
        >
          <svg className="w-6 h-6" fill="none" stroke="currentColor" viewBox="0 0 24 24">
            {isOpen ? (
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M6 18L18 6M6 6l12 12" />
            ) : (
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M4 6h16M4 12h16M4 18h16" />
            )}
          </svg>
        </button>
      </div>

      {/* Mobile menu */}
      {isOpen && (
        <div className="md:hidden border-t border-navy/10 bg-cream">
          {links.map((link) => (
            <Link
              key={link.href}
              href={link.href}
              className="block px-4 py-3 text-navy hover:bg-navy/5 transition-colors"
              onClick={() => setIsOpen(false)}
            >
              {link.label}
            </Link>
          ))}
        </div>
      )}
    </nav>
  );
}
```
